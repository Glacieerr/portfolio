当前阶段如何使用

          
    
            
  1. 点击「新增作品」，填写作品信息。
    2. 
            
  2. 点击「保存到本地列表」，把作品保存进当前 CMS 数据。
    3. 
            
  3. 点击「JSON 预览」检查生成的数据结构。
    4. 
            
  4. 输入 Admin Key，并点击「保存密钥」。
    5. 
            
  5. 点击「发布到 GitHub」，自动更新 GitHub 仓库里的 data/works.json。
    6. 
            
  6. 等待 Vercel 自动部署完成后，前台页面会读取最新作品数据。
    7. 
          


          

            注意：Admin Key 是 Vercel 环境变量里的 ADMIN_KEY，
            不是 GitHub Token。GitHub Token 只应保存在 Vercel Environment Variables 中。
          


          

            如果只想手动备份数据，也可以点击「导出 works.json」下载当前 JSON 文件。
